# Fitness-Calculator

If you want to increase muscle mass or lose some weight , this program is maded to calculate how much protein,carbohydrates,calories,fats you need to eat per day .
Just you need to follow the steps :

1 - Download the rar file "Download-Program"
2 - Make some random folder in your computer and extract all files from rar file in the folder
3 - You need to have Visual Studio to start the project , because is made on simple console app ( soon will be updated on other technologie ) 
4 - Open the "FitnessCalculator.csproj" file , the files will be started automatically in visual studio
5 - When the files are started in Visual Studio , just press CTRL + F5 , the program will start
6 - Follow the steps in the program , write your name , age and weight currect ! 
7 - Choice from the menu in the program what you want to do , increase , lose or maintenance your weight
8 - If you follow this steps currectly , the program should automatically calculates how much you need protein,carbohydrates,calories,fats per day

IMPORTANT : write currectly your information and choices in menu , because if you don't write something currectly you must start the program again with CTRL+F5.


ENJOY IT!
